---
title: "Get Plan"
api: "GET /plans/{planId}"
description: "Retrieve details for a specific subscription plan by its ID."
authMethod: "bearer"
---

## Path Parameters

<ParamField path="planId" type="string" required>
  The numeric ID of the plan. Retrieve plan IDs from the List Plans endpoint.
</ParamField>

<RequestExample>

```bash cURL
curl -X GET https://api.voxmind.ai/plans/1 \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Accept: application/json"
```

</RequestExample>

<ResponseExample>

```json 200 OK
{
  "id": 1,
  "details": {
    "name": "Starter",
    "monthly_calls_included": 2500,
    "overage_rate_gbp": 0.18
  },
  "created_at": "2025-01-01T00:00:00Z",
  "updated_at": "2025-01-01T00:00:00Z"
}
```

</ResponseExample>
