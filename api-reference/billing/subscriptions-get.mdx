api: "GET /organisations/{orgId}/subscriptions/{subscriptionId}"
description: "Retrieve a specific subscription record by its ID."
authMethod: "bearer"
---

## Path Parameters

<ParamField path="orgId" type="string" required>
  Your organisation's unique identifier.
</ParamField>

<ParamField path="subscriptionId" type="string" required>
  The numeric ID of the subscription record.
</ParamField>

<RequestExample>

```bash cURL
curl -X GET https://api.voxmind.ai/organisations/42/subscriptions/3 \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Accept: application/json"
```

</RequestExample>
