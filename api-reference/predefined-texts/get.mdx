---
title: "Get Predefined Text"
api: "GET /organisations/{orgId}/predefined-texts/{textId}"
description: "Retrieve a specific predefined text by its ID."
authMethod: "bearer"
---

## Path Parameters

<ParamField path="orgId" type="string" required>
  Your organisation's unique identifier.
</ParamField>

<ParamField path="textId" type="string" required>
  The numeric ID of the predefined text record.
</ParamField>

<RequestExample>

```bash cURL
curl -X GET https://api.voxmind.ai/organisations/42/predefined-texts/23 \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Accept: application/json"
```

</RequestExample>

<ResponseExample>

```json 200 OK
{
  "id": 23,
  "org_id": 42,
  "language": "en-UK",
  "text_code": 1,
  "text": "Please say your full name to register your voice.",
  "created_at": "2025-03-15T10:00:00Z",
  "updated_at": "2025-03-15T10:00:00Z"
}
```

</ResponseExample>
